---
description: Coding workflow - Explore > Plan > Code > Verify
---
1. Skill gate (tdd/debugging/code-review). 2. Explore (read files, grep). 3. Plan (state changes + risks).
4. Implement (dispatch @forge if non-trivial). 5. Verify (tests+lint, show output). 6. Report.
Never: skip explore, modify unread files, claim done without evidence, self-correct >2x.
